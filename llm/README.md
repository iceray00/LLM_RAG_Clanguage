

## Download Ollama

```bash
curl -fsSL https://ollama.com/install.sh | sh
```


```bash
ollama serve
ollama run qwen2
```
- Default install `Qwen2-7B-int4`


## Quick Start

In one terminal, exec:
```bash
ollama serve
```

in other terminal, use `main_test.py`:
```bash
python3 main_test.py
```
